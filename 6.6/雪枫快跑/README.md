# 雪枫跑酷（HTML5）

这是一个**2D横版（天天酷跑式）**的跑酷小游戏：角色“雪枫”自动向前奔跑，你只需要用**跳跃/下滑**来躲避“雪堆”“考试卷”等障碍，拾取“巧乐兹雪糕”“雪碧”等道具获得增益。角色有 **3条血**，支持**二段跳**，并提供**三种难度**（不同前进速度）。

## 运行方式

### 方式A：直接打开（最简单）
双击打开 `index.html` 即可运行。

> 如果你的浏览器对本地文件的语音/资源有限制，推荐用 Chrome/Edge 打开。

### 方式B：本地小服务器（更稳）
如果你装了 Node.js：
```bash
npx serve .
```
然后按提示在浏览器打开网址即可。

## 操作
- 电脑：空格/↑ 跳跃（二段跳）；↓ 下滑/铲
- 手机：点“跳”跳跃（二段跳）；点“滑”下滑/铲（建议横屏/全屏）

## 道具效果（已实现）
- **巧乐兹雪糕**：拾取后角色语音说“你跑不过我你信吗”，并在一段时间内 **加速 + 双倍得分**
- **雪碧**：拾取后角色语音说“跑起来”，并获得 **5秒无敌**

> 时长与倍率可在 `game.js` 顶部 `CONFIG.itemEffects` 里调整。

## 障碍与掉血（已实现）
- 障碍：**雪堆**、**考试卷**
- 规则（天天酷跑式）：
  - **地面障碍**：需要**跳**过去，否则会撞到掉血
  - **低空障碍**：跳起来容易撞到，建议用**下滑/铲**从下面钻过去（下滑会缩小碰撞盒）
- 生命：3条血，碰到有效障碍会掉血，掉到0结束

## 如何替换你提供的素材（PNG）

把你的图片放到 `assets/` 目录下，并使用以下文件名（建议透明背景 PNG）：

- `assets/xuefeng.png`（角色，建议 64×64 或 128×128）
- `assets/icecream.png`（巧乐兹雪糕）
- `assets/sprite.png`（雪碧）
- `assets/snowpile.png`（雪堆）
- `assets/exam.png`（考试卷）

代码会**优先加载这些图片**；如果缺失，就会自动使用占位贴图（保证能运行）。

## 可改的玩法参数
打开 `game.js` 顶部 `CONFIG`：
- `difficulties.easy/normal/hard.speed`：不同难度的前进速度
- `difficulties.*.spawnEveryMs`：生成频率（越小越密集）
- `player.jumpVelocity / gravity`：跳跃手感
- `itemEffects.icecream.durationMs / speedMultiplier / scoreMultiplier`：巧乐兹效果
- `itemEffects.sprite.durationMs`：无敌时长
